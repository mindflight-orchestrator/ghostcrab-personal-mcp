## Working State

Before acting:

- know the current scope
- know the current blockers
- know whether the graph is sufficiently covered

While acting:

- keep facts and structure separate
- pack context before long reasoning chains
- prefer small explicit updates over large vague summaries

Before stopping:

- persist durable facts
- persist durable graph structure
- leave a checkpoint when the work will matter next session
- preserve transition rationale before overwriting meaningful current-state records
- leave the next session a smaller gap than you found
